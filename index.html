3D Reconstruction
One of the leading technology is a 3D reconstruction which has been challenging at very points, in this report we will talk about a system built on the python programming language to generate a 3D model from stereo vision. The system uses any regular camera which in the first phase gets calibrated extracting camera information like focal length distortions in the camera, intrinsic and extrinsic parameters of the camera, etc. In phase 2 we will use that parameter to create a disparity map and using that map we will generate a 3D cloud which will we transfer into a 3D point cloud using a transformation matrix and reprojecting points into 3D and creating a point cloud. Finally, we will use MeshLab to visualize our 3D model. Keywords: 3D reconstruction; 3D model; camera calibration; depth map; point cloud Python programming language.
